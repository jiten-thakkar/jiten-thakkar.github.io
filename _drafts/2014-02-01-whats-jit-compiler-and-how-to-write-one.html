Compiler is a programme that translates programming code from one form to another. It might be from higher order language or intermediate representation code(known as bytecode) to machine code or it can be from one language to another. Normally we use them to generate executable file that we can run on our computer. The compilers that generate executable files to be run later  on are called Ahead-of-Time(AOT) compilers or simply compiler. These are the compilers that we commonly use for languages like C,C++ etc. On the other hand, Just-In-Time(JIT) compiler converts the code in runtime. The feeding code would be a continuous stream of bytecode. The work of converting the language code to bytecode is left to AOT type of compilers. For example, compiler for Java programming language (for example 'javac' from Oracle) generates a .class file (Java Class File) which contains Java Bytecode. This .class file is later run in a Java Virtual Machine(JVM) which feeds these bytcodes in .class file to a JIT compiler. It generates and executes machine code as it goes on.
